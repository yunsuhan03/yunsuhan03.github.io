# Cafe Review Analysis Dashboard

## Project Overview

This dashboard analyzes cafe review data and provides interactive visualizations to help users compare cafes based on ratings, review counts, themes, and customer preferences.

## Features

- District Filter
- Theme Filter
- Cafe Search
- Rating Analysis
- Theme Distribution
- Keyword Analysis
- Top Reviewed Cafes

## Technologies

- Python
- Streamlit
- Pandas
- Plotly
- GitHub

## Run Application

streamlit run app.py
