☕ Cafe Review Analysis Dashboard

Project Overview

This dashboard provides interactive visualizations for analyzing cafe ratings, review counts, themes, and customer preferences.

Dashboard Features

  • District Filtering
  • Theme Filtering
  • Cafe Search
  • Rating vs Review Count Analysis
  • Theme Distribution
  • Keyword Analysis
  • Top Reviewed Cafes
  • Dashboard Insights

Technology Stack

  • Python
  • Streamlit
  • Pandas
  • Plotly
  • GitHub

Run Application

streamlit run app.py
